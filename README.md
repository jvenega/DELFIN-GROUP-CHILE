# 🐬 Delfin Group Chile – Repositorio de Desarrollos

## 📌 Descripción

Este repositorio centraliza los desarrollos realizados para **Delfin Group Chile**, incluyendo aplicaciones, módulos y componentes tanto de **frontend (React + Vite)** como de **backend (Node.js)**.

Su objetivo es mantener una base organizada, escalable y reutilizable de soluciones tecnológicas utilizadas dentro de la empresa.

---

## ⚙️ Tecnologías Utilizadas

### Frontend

* React
* Vite
* JavaScript / TypeScript

### Backend

* Node.js
* Express (u otros frameworks según módulo)

---


## 📁 Buenas Prácticas

* Uso de `.gitignore` por módulo cuando sea necesario
* Separación clara entre frontend y backend
* Manejo de variables de entorno (`.env`)
* Código modular y reutilizable
